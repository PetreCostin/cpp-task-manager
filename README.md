# 🖥️ C++ Task Manager (TUI)

A terminal-based task manager built with C++ and ncurses.

## 🚀 Features
- Interactive UI (keyboard navigation)
- Add tasks
- Delete tasks
- Real-time rendering

## 🎮 Controls
- ↑ ↓ : Navigate
- ENTER : Select
- a : Add task
- d : Delete task
- q : Quit
